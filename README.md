# Taskify - To Do List

A simple Todo List web app built with React.  
This project demonstrates basic **CRUD operations** (Create, Read, Update, Delete) with a UI.

## Features
- Add new tasks
- Edit existing tasks
- Mark tasks as complete
- Delete tasks

## Tech Stack
- **Frontend:** React
- **Styling:** CSS
- **Backend:** Nest.js, MySQL
